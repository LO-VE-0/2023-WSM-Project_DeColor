# .DeColor

<br />

## 소개
 디자인적 모호함을 방지하기 위해, 디자이너와 개발자들은 색각 이상자들의 시선을 고려하여 색상을 적절히 선택해야 합니다. 그러기 위해서, 색약인을 고려한 색을 선택하도록 도와줄 서비스가 필요합니다.
<br />
**DeColor는 색각이상자를 고려한 컬러 팔레트를 추천하고 디자인 팁을 공유하는 웹사이트입니다.**
